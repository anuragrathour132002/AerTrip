## 🛠️ Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/anuragrathour132002/AerTrip
cd AerTrip

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev

# Open in browser at: http://localhost:5173
